BioSimSpace.IO¶

The IO package contains tools for reading and writing molecular systems.

Functions¶

fileFormats() Return a list of the supported formats.
formatInfo(format) Return information for the specified file format.
readMolecules(files[, property_map]) Read a molecular system from file.
readPDB(id[, property_map]) Read a molecular system from a Protein Data Bank (PDBP) ID in the RSCB PDB website.
saveMolecules(filebase, system, fileformat) Save a molecular system to file.
