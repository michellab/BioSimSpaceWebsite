BioSimSpace.Gateway.Area¶

class BioSimSpace.Gateway.Area(help=None, default=None, unit=None, minimum=None, maximum=None, allowed=None)¶

An area requirement.

Examples

Create an area requirement with a default of 10 square Angstrom.

>>> import BioSimSpace as BSS
>>> my_area = BSS.Gateway.Area(help="An area requirement", default=10, unit="angstrom2")

The same, but explicitly passing a Area for the default.

>>> import BioSimSpace as BSS
>>> my_area = BSS.Gateway.Area(help="An area requirement", default=10*BSS.Units.Area.angstrom2)

Create an area requirement with a default of 10 square Angstrom and a maximum of 50 square nanometers. Note that the unit is taken from the default value.

>>> import BioSimSpace as BSS
>>> my_area = BSS.Gateway.Area(help="An area requirement",
...                            default=100*BSS.Units.Area.angstrom2,
...                            maximum=50*BSS.Units.Area.nanometer2)
