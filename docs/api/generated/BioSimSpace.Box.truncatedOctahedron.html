BioSimSpace.Box.truncatedOctahedron¶

BioSimSpace.Box.truncatedOctahedron(image_distance)¶

Generate parameters for a truncated octahedron.

Parameters

image_distance (Length) – The image distance.

Returns

  • box ([Length]) – The box vector magnitudes.

  • angles ([Angle]) – The box vector angles: yz, xz, and xy.
