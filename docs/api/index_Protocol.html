BioSimSpace.Protocol¶

The Protocol package contains tools to select features from MD-trajectories. It also assigns them to a discrete state space, which will be later used in Markov modeling.

Functionality for defining simulation protocols. Author: Lester Hedges <lester.hedges@gmail.com>

BioSimSpace.Protocol.createProtocol(protocol)¶

Create a default simulation protocol.

Parameters:protocol (str) – The name of the protocol.
BioSimSpace.Protocol.protocols()¶

Return a list of the supported Molecular Dynamics protocols.
