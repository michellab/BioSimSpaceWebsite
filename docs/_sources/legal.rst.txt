Usage tracking
==============

BioSimSpace has in-built analytics to track usage. To disable, simply set the
following environment variable, e.g. in your ``~/.bashrc``:

.. code-block:: bash

   export SIRE_DONT_PHONEHOME=1

By default, you will see exactly what information is recorded. To silence this
message:

.. code-block:: bash

   export SIRE_SILENT_PHONEHOME=1
